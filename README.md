## 🛠️获取源码/Source Code

要构建此项目，您需要将项目源码克隆到本地。

您可以使用 Git 命令行：

```powershell
git clone https://github.com/SIXiaolong1117/SeeMyServer.git
```

或者更方便的，使用 Visual Studio 的“克隆存储库”克隆本仓库。

使用 Visual Studio 打开项目根目录的 `SeeMyServer.sln`，即可进行调试和打包。
